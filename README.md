# 📌 Project Overview

This project presents an **end-to-end Time Series Forecasting System** developed to predict future sales using multiple statistical, machine learning, and deep learning models.

The project compares six forecasting approaches and visualizes their performance through an interactive **Power BI Dashboard**.

The complete workflow includes:

- Data Collection
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Building
- Forecast Generation
- Model Evaluation
- Interactive Dashboard
- Performance Comparison

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Madhuri-Kumari2504/Time-Series-Forecasting.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

# ▶️ How to Run

1. Download the dataset.
2. Place it inside the **data/** folder.
3. Run preprocessing notebook.
4. Train forecasting models.
5. Save trained models.
6. Open the Power BI dashboard.
7. Explore forecasts.
